# YouTube Homepage Clone 🚀

A fully responsive front-end clone of the YouTube Homepage, built from scratch to practice advanced structural layouts and responsive web design.

## 🔗 Project Links
- **Live Demo:** [https://fariyaahmedosd.github.io/YouTube-Clone/](https://fariyaahmedosd.github.io/YouTube-Clone/)
- **Source Code:** [https://github.com/FariyaAhmedosd/YouTube-Clone](https://github.com/FariyaAhmedosd/YouTube-Clone)

## 🛠️ Tech Stack Used
- **HTML5:** For structuring the nested layout, navigation, sidebar, and video grid.
- **CSS3:** For raw custom styling, layouts, layouts, and animations.

## 🧠 Key Features & What I Learned
- **Nested Layout & Nested Flexbox:** Implemented complex nested structures, combining CSS Grid for the main layout and nested flexbox containers to perfectly align row items, headers, and video details.
- **Positioning & Layout Control:** Mastered the use of `absolute`, `relative`, and `fixed` positions to keep the top navbar and left sidebar pinned during scrolling.
- **Transitions & Hover Effects:** Created smooth transition effects for subtle button reactions, sidebar link changes, and video card scaling.
- **Custom Tooltips:** Implemented pure CSS tooltips for quick, user-friendly icon hints during hover interactions.
- **Text Styling & Typography:** Managed text-overflow, line clamps, and specific font weighting to replicate YouTube's accurate typography.
- **Responsive Web Design:** Used media queries to ensure smooth layout tracking and responsiveness across different device screen sizes.

# YouTube Homepage Clone 🚀

A fully responsive front-end clone of the YouTube Homepage, built from scratch to practice advanced structural layouts and responsive web design.

## 🔗 Project Links
- **Live Demo:** [Tomar GitHub Pages Link Ekhane Dibe]
- **Source Code:** [Tomar GitHub Repository Link Ekhane Dibe]

## 🛠️ Tech Stack Used
- **HTML5:** For structuring the layout (Navigation, Sidebar, Video Grid, and Cards).
- **CSS3:** For modern styling, custom grid systems, and animations.

## 🧠 Key Features & What I Learned
- **CSS Grid & Flexbox:** Implemented a robust grid layout for the video thumbnails and used flexbox for aligning the top navigation items perfectly.
- **Responsive Design:** Used standard media queries to ensure the sidebar hides and the video layout transitions smoothly from desktop screens to mobile screens.
- **Hover Effects:** Added interactive hover transitions on buttons, thumbnails, and links for a more dynamic user experience.
